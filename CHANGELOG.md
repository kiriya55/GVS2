# 截至目前的修改内容总结

- 字幕圈选区域修改：改为单个可调整红框。
- 字幕事件呈现显示修改：改为下拉形式可看到字幕事件对应内容。
- 字幕样式导入功能：支持通过已有ASS文件导入样式并自动匹配结构化样式描述
- 优化benchmark导入逻辑。
- 改进预提取帧：预提取阶段增加逐条日志和 UI 进度，避免看起来像卡死。
- 增加抽帧并发：新增独立“抽帧并发数”设置，默认 5，上限 12，不复用 API 并发数设定。
- 调整样式匹配策略：匹配不上改为套用 `Default`；“需核查”保留疑似样式，并给当前字幕加黄色斜体醒目标记。
- 修复文本解析失败：允许模型返回 JSON 字符串里带未转义换行和`\r`，解析后转换为 ASS 的 `\N`。
